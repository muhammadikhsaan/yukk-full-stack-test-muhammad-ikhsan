### Specification
- PostgreSQL database
  - can install use docker-compose file in this project
- PHP 8.1.x
- Laravel 10
- React JS
- Inertia
- Tailwind

### How To Start
- Open Terminal
- Run `composer install`
- Run `npm install`
- Setup `.env` file
- Run `php artisan migrate`
- Run `php artisan serve`
- Run `npm run dev`
- Open `localhost:8000` on browser
